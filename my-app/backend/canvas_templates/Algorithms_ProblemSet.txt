Problem 1: Markov Decision Processes (25 points)
Consider a simple MDP with 3 states {S1, S2, S3} and 2 actions {A, B}.

a) Define the transition probabilities:
   P(S2|S1,A) = 0.7, P(S3|S1,A) = 0.3
   P(S1|S2,B) = 0.5, P(S3|S2,B) = 0.5
   
b) Given rewards: R(S1) = 10, R(S2) = 5, R(S3) = 20
   Calculate expected value of being in S1 and taking action A
   
c) Use value iteration with γ = 0.9 for 3 iterations starting with V₀(s) = 0

Problem 2: Monte Carlo Tree Search (25 points)

   Calculate UCB1 values for two nodes:
   - Node 1: wins=15, visits=20, parent visits=50
   - Node 2: wins=8, visits=10, parent visits=50
   Use C = √2
